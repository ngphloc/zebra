Thank you for your interest in WOW! 1.0 as your adaptive Web-based system. You are now viewing the main starting page of the WOW! distribution. From this page you can go to the following sub-parts:

  • The configuration link leads to an interface for manipulating the different WOW! data structures. It lets you choose between the use of XML files or a mySQL database for the concept structures and the user models, and lets you convert authoring formats to internal formats. It lets you define authors and assign applications to authors.
  • The Application Management Tool is the main authoring tool. It lets you transfer files from your local machine to the server and back, and lets you start most authoring tools by simply clicking on the names of the authoring files.
  • In the Author Workplace you can change your settings as an author, and access all the authoring tools. Authorization is required to change your author settings. You can also access the authoring tools directly from the The authoring page. (The tools ask for authorization when you start them.)
