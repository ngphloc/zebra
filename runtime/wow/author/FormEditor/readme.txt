FormEditor v1.0
---------------
* Source Code: bevat the java-files van de FormEditor, behalve die voor het Servlet-gedeelte
* Compiled: bevat de gecompileerde code, als ook de nodige DTD's, gifs, xmlfiles en htmlfiles. Deze is getest met de jswdk-1.0 en zou ook in een map onder de jswdk-1.0 moeten komen. Nadat de server is gestart, kan men de FormEditor openen door naar de file 'test.html' te browsen. Deze bevat namelijk het applet.
* Servlet: bevat de java-files en gecompileerde code voor de 3 servletjes (saven van form, serveren van form en processing van input komende van form). Ik had die onder 'servlet' staan in de jswdk-1.0. In deze files staan een paar directe verwijzingen naar 'localhost....' die eventueel nog gewijzigd moeten worden.
* WowFormTags: bevat de DTD's voor XHTML. Hiermee wordt voorlopig nog niet gewerkt.
