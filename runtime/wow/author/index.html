The WOW! system offers the following authoring tools:

  • The Form Editor lets you create forms through which users can change parts of their user model.
  • The Concept Editor lets you create concepts, attributes and condition-action rules. This is the most powerful but also the lowest level authoring tool.
  • The Graph Authoring Tool lets you create concepts and concept relationships. This is an easy to use, high level authoring tool.
  • The Bayes Authoring Tool lets you load and modify Bayesian network for supporting user knowledge inference.
  • The Test Editor lets you create multiple-choice tests.
  • The Convert MOT to WOW! form lets you convert data from MOT and create an WOW! course without any intermediate steps.
  • The Import MOT form lets you import data from a MOT database and store it as a CAF file.
  • The Extract CAF form lets you extract a CAF file and apply a strategy to generate a new course.
