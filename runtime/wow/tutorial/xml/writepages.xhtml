Write Pages

The application in itself not only constitutes of the conceptual structure, but also pages written in a specific language which represent the various concepts used.

A model set of pages is created for your reference. The WOW! distribution contains a directory "example" with a set of pages to be used for the example application. These pages are actually in HTML, not XHTML, but WOW! automatically converts HTML to XHTML in order to be able to process it.

For further reference on how to create pages check out the page format section of the tutorial.
