Visibility/Presentation of Fragments

Sometimes you may wish to conditionally emphasize, deemphasize or even hide a fragment of text. Instead of conditionally including different objects in order to obtain this effect, you can control the presentation of a <span> item through the visibility attribute of a concept:

  • A fragment is conditionally presented in different ways by putting a <span> tag around it as follows:
    <span id="tutorial.fragmentconcept" class="conditional">The presentation style of this sentence may vary.</span>
  • The integer attribute visibility of the concept tutorial.fragmentconcept determines how the text is presented. Currently the values that have a meaning are multiples of 10, between 0 and 100.
  • When the visibility is 0 the fragment is not shown at all. It is included in the page as a comment, so it can be seen when viewing the html source of the page.
  • When the visibility is 10 a tooltip is presented. The icon for this must be supplied by the application, so you can use your own favorite icon. Also, a piece of Javascript code must be placed in a file that is included in the page: 
     function showTip(img, tip)
 { tiptext = document.getElementById(tip);
   tiptext.style.visibility = 'visible';
   tipimg = document.getElementById(img);
   tiptext.style.left = tipimg.offsetLeft + 25;
   teptext.style.top = tipimg.offsetTop;
 }
 function hideTip(tip)
 { tiptext = document.getElementById(tip);
   tiptext.style.visibility = 'hidden';
 }
  • The presentation of other values can be defined in an external style sheet. The "conditional" class of the <span> tag is replaced by a class "cN" where N is the visibility value. (The "c" is added because class values must be strings, not numbers.) An example of a possible style sheet is:
    .c20	{font-size:smaller;}
.c30	{color: #808080; background-color: #EEEEEE;}
.c40	{font-style: normal;}
.c50	{font-size:larger;}
.c60	{font-style: italic;}
.c70	{text-decoration: underline;}
.c80	{text-transform: uppercase;}
.c90	{font-weight: bold;}
.c100	{color: #FFFFFF; background-color: #5F9EA0;}
