Advanced Topics for System Designers and Server Managers

Authoring for WOW! can be done by editing concept relationships through the graph author. When the graph author seems too restricted you can choose to edit the concept structure and event-condition-action rules through the concept editor or to edit the concepts and rules directly in the corresponding XML file. However, the functionality of the graph author can be stretched by creating new templates for concept relationship types and/or changing existing ones.

Another potential advanced aspect is the performance. When the performance of an WOW! server is too low there are several ways to try to improve it by changing the WOW! configuration:

  • When using XML files instead of the mySQL database performance can be increased by moving the directory tree with the files to a separate disk (through the XMLROOT parameter). Note that moving the files to a different file system on the same disk does not help.
  • When using database server one can move the database to a different machine than the one running the WOW! server, and change the "dbhost" parameter to refer to that machine.
