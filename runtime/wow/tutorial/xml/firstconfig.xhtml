Configuring WOW!

We assume that you have built the WOW! binaries, and that the TomCat server (for instance Sun's version JWDSP from http://java.sun.com/webservices/downloads/webservicespack.html) is installed and running, and that you have created a "context" for WOW!, with "/wow" as pathname.

Configuring WOW! consists of the following steps:

  1. In the browser you go to the address /wow/Config. The first time you do this a page appears that says "Installing Configuration file". This step actually executes a servlet that creates the WOW! configuration files containing the information you entered when creating the WOW! context for TomCat.
  2. Because the first step overwrites the initial configuration file, the next step only works if you configured the WOW! context to be reloadable. If you click on the Configuration link you go to "WowConfig" page. If the context is not reloadable you have to restart the TomCat server first. (Execute shutdown.sh and startup.sh on Linux or shutdown.bat and startup.bat on Windows.)
    You can always go to the "WowConfig" page later, using the link /wow/Config. You will never see the "Installing Configuration file" page again.
  3. When you try to go to WowConfig page you are prompted for a login and password. Initially you should use "wow" as login and no password.
  4. The first thing to do when shown the WowConfig page is to select "Manager Configuration" and then select "Change an existing user" from the Manager administration menu, in order to change the "wow" login to a name you prefer, and to add a password. This is important for security reasons, because everyone who knows WOW! also knows that it comes with "wow" as the default manager without a password.
  5. Through the Author menu you can create authors and assign applications to authors.
  6. WOW! is now ready to be used. It contains this tutorial as an example adaptive hyperdocument. It uses a set of XML files to store the concept information and the user profiles. To change the WOW! database configuration, you can click on the "Configure Database" link to create the database. Note that your database must have been configured to accept the login and password that you choose, before attempting to have WOW! create the database.
