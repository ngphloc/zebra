Further explanation

The list that comes with this WOW! release and tutorial contains a number of concept relationship types for illustration purposes, as well as some commonly used ones like "knowledge_propagation", "knowledge update" and "prerequisite". The list is extracted from a list of templates. You can create your own templates, but that requires a thorough knowledge of the WOW! adaptation rule language.

For further reference, you can visit the Graph Author page of the tutorial.
