Defining Layout of the Application

Most adaptive applications do not only show pages with links, but have some additional structures, like the "menu" you see on the left in this tutorial. Each WOW! application defines its own look and feel. This consists of a definition of layout, consisting of html frames and which information goes into which frame, and a specification of concept presentation.

All this is specified with the help of the following files:

  • Layoutconfig.xml
  • ConceptTypeConfig.xml
  • header.xhtml (or/and) footer.xhtml (OPTIONAL)

You can create your own definitions for layout or use the existing definitions provided with the system. To use the ones provided with system, copy the above given files from /wow/tutorial to /wow/example and change every occurence of the word 'tutorial' with 'example'.

For further reference see the corresponding page of the tutorial: The Layout of an WOW! Application.

Note: We will also need a login page and a registration page for our application. We can also copy these from the directory /wow/tutorial and modify them for our use (changing at least every occurence of "tutorial" into "example").
