The Graph Author Tool

The Graph Author is the main authoring tool for the concept structure. It lets you create a concept hierarchy and concept relationships of different types. We briefly describe the use of the Graph Author here, using a series of screen dumps to guide you through the process.

  • When you start the Graph Author it first asks for a login and password. (If you start it from the application management tool you do not have to authenticate yourself in the graph author as you already did that in AMT.) As of WOW! 2.0 authors are identified and have separate work spaces. The manager can add new authors, either as part of the initial installation or at a later time using the configuration menu.
  • After logging in the main Graph Author window appears. It consists of four main parts:
    • Top left: menus and buttons. The functions of WOW are all available through the ten buttons
      Create a new WOW! application. You are asked for a confirmation if this action would wipe out edits in the application you are editing.
      Open an WOW! application. Through a pop-up dialog you can select one of the applications for which you are an author.
